## Requirements

- Python 3.7
- Python 3 PIP 18.1
- Virtualenv 16.7 or Pygame 1.9

## Setup with virtualenv

```shell
cd make-moose-move-out/
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run

```shell
python main.py
```
